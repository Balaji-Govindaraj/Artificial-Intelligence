Download the dataset from : https://www.kaggle.com/c/dogs-vs-cats/data
Create folders for training train/dogs and train/cats
Create a folder test paste all the test image in the folder
Run the code using the following command:
	python main.py train
	python main.py test

I have also create a logs for tensorboard visualization for loss calculation.
To see the loss in tensorboard, run the following command.
	go the CNN directory
	tensorboard --logdir logs
